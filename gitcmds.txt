1 git init cmdchallenge1
initialize an existing directory as a Git repository

2 git status  
show modified files in working directory, staged for your next commit

3 git add . and git add Readme.md 
add a file as it looks now to your next commit (stage)

4 git commit -m 'mod1'
git commit will create a snapshot of the changes and save it to the git directory.

5 git branch
makes a branch folder

6 git checkout branchname
to switch branches

7 git remote add origin url
add a git URL as an alias

8 git remote -v
to verify new remote url

9 git push HEAD:master
to push commits on git

10 git help
to get help commands

11 git push [alias] [branch]
Transmit local branch commits to the remote repository branch

12 git pull
fetch and merge any commits from the tracking remote branch

13 git rm [file]
delete the file from project and stage the removal for commit

14 git mv [existing-path] [new-path]
change an existing file path and stage the move

15 git rebase [branch]
apply any commits of current branch ahead of specified one
